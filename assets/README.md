# Assets

Placer ici les fichiers requis par Expo :
- `icon.png` (1024x1024) — icône de l'app
- `splash.png` (1284x2778) — écran de splash
- `adaptive-icon.png` (1024x1024) — icône adaptive Android
- `favicon.png` (48x48) — favicon web (dev uniquement)

Pour le MVP, utiliser des placeholder générés via :
- https://placeholdermaker.com
- ou simplement un carré vert #22C55E avec le texte "ST"
